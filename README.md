# ComponentInserter
Put the .rbxmx file in your plugins folder

Allows you to easily insert StateComponents into your InfiltrationEngine custom mission, as well as view brief but useful descriptions for listed StateComponents and their attributes

Inserting StateComponents will not work if there is not a DebugMission folder in the Workspace

Activate the plugin to create a widget containing a list of StateComponents
<img width="383" height="607" alt="image" src="https://github.com/user-attachments/assets/bdf7898a-960e-46c1-8448-0e938d5373b5" />

You can use the search box to filter for specific StateComponents
<img width="379" height="605" alt="image" src="https://github.com/user-attachments/assets/55de3508-133b-49c8-9353-f9ba5ae0a2e3" />


You can click on the name of a StateComponent to insert it into the StateComponents folder of your MissionSetup folder
<img width="631" height="262" alt="image" src="https://github.com/user-attachments/assets/ae405587-6956-489f-9d34-af8a82c3edde" />


You can press the "?" button next to a StateComponent's name in the list to view useful info about it and its attributes
<img width="380" height="604" alt="image" src="https://github.com/user-attachments/assets/9b3ef38c-1b75-4b53-9ed3-c0afb29137e0" />
